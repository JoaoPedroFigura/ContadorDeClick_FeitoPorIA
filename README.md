# Contador de Cliques Web

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de explorar a capacidade de uma Inteligência Artificial de gerar código a partir de instruções em linguagem natural. A aplicação consiste em um contador de cliques web integrado a um temporizador, servindo como experimento prático para avaliar o potencial da IA no desenvolvimento de software.

## Funcionalidades

- Contagem de cliques em tempo real
- Temporizador integrado
- Interface web simples e intuitiva
- Experimento de desenvolvimento assistido por IA

## Tecnologias utilizadas

- Claude Code
- HTML
- CSS
- JavaScript

## Objetivo

O principal objetivo deste projeto é estudar como ferramentas de Inteligência Artificial podem auxiliar no processo de desenvolvimento, transformando prompts em código funcional.
